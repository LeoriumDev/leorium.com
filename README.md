# leorium.com

Personal site and blog. Built with [Hugo](https://gohugo.io/) + [Blowfish](https://blowfish.page/). Deployed on [Cloudflare Pages](https://pages.cloudflare.com/).

## Local Development

```bash
hugo server -D
```

## Structure

- `content/blog/` — blog posts and series
- `content/projects/` — project pages
- `content/now/` — /now page
- `assets/css/custom.css` — custom styles
- `layouts/partials/` — theme overrides
